# Fotos de jugadores para el Quinteto ideal

Coloca aquí las fotos de los jugadores para que aparezcan en la imagen
descargable del quinteto (quinteto.html → Descargar / Compartir).

- **Nombre del fichero**: el `id` del jugador + `.jpg` (p. ej. `pau-gasol.jpg`).
- Formato JPG (o cambia la extensión en `qntBuildImage` si usas otra).
- Recomendado cuadradas ~400×400 px (se recortan en círculo).
- Deben estar en este repo (mismo origen): las fotos externas de
  basketball-reference NO sirven para la imagen porque bloquean CORS.

Si un jugador no tiene foto aquí, en la imagen sale con sus iniciales.
